Les projets que j'ai réalisés

Projets dans le cadre du BUT Informatique

Dans le cadre du BUT Informatique, j'ai eu l'opportunité de réaliser différents projets touchant à divers domaines de l'informatique. De la programmation en C à l'analyse économique, écologique et numérique d'une entreprise, en passant par la création d'un site web répondant aux demandes d'un client. Ces projets m'ont permis de développer des compétences utiles et concrètes dans le domaine de l'informatique.

Le jeu du Snake

Programmation du jeu Snake de manière incrémentale. Plusieurs versions du programme ont été développées, chacune enrichissant la précédente.

Le Snake autonome

Programmation du jeu Snake afin de rendre le serpent autonome, c'est-à-dire qu'il se déplace automatiquement pour manger les pommes le plus efficacement possible. Différentes versions ont été développées, chacune ajoutant de nouvelles fonctionnalités.

Systèmes informatiques

Réalisation d'une chaîne de traitement de données en Bash et PHP, avec Docker, pour remplir le cahier des charges d'une équipe de développement web sur les Jeux Olympiques.

Gestion d'une ligue de football

Conception et modélisation d'une base de données avec un diagramme de classes. Traduction du diagramme vers un modèle relationnel puis implantation en Tutorial D.

Site web sur les JO

Réalisation d'un site web sur le thème des Jeux Olympiques en groupe avec une procédure de conception rigoureuse.

Analyse de Coca-Cola

Étude de l'impact économique et écologique de l'entreprise Coca-Cola à travers une analyse approfondie de ses pratiques et de son modèle économique.
