Site Web sur les Jeux Olympiques

Le but de ce projet est de créer un site web informatif et interactif, en groupe, autour des Jeux Olympiques, en mettant en avant l’histoire, les disciplines, les athlètes emblématiques et les actualités.

Compétence n°5 : Implémentation d'un besoin client

Ce projet est en lien avec la compétence 1 du programme du BUT informatique : développer des applications informatiques simples.

Les outils utilisés :

  • HTML (structure des pages web)
  • CSS (mise en forme et design)
  • JavaScript (interactivité du site)
  • Figma (maquettage et conception UI)
  • GitHub (gestion de version et collaboration)
  • Google Sheets (analyse de données sportives)
  • Visual Studio Code (édition de code)

Les étapes du projet

Le développement du site s’est déroulé en plusieurs phases. Cliquez sur chaque étape pour voir les détails.

Recueil des attentes du client : proposer un site accessible et responsive dédié aux Jeux Olympiques.

Élaboration de la structure du site : page d'accueil, sports, athlètes, disciplines.

Définition de la palette de couleurs, choix des typographies et de l'identité visuelle.

Création de maquettes interactives sur Figma afin de valider le design avec le client avant l'implémentation.

Intégration des pages avec HTML/CSS/JS, ajout de contenus dynamiques, responsive design, et tests sur différents navigateurs.
