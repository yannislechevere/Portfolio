SWOT et RSE de Coca Cola

Le but de ce projet est de réaliser le SWOT (Strengths, Weaknesses, Opportunities et Threats) et la RSE (Responsabilité Sociétale des Entreprises) de l'entreprise Coca Cola.

Compétance n°6 : Découverte de l’environnement économique et écologique

Ce projet est en lien avec la competance 6 du programme du BUT informatique : Acquérir, développer et exploiter les aptitudes nécessaires pour travailler efficacement dans une équipe informatique.

Les outils utiliser :

  • Google Docs (pour rediger des comptes rendus)
  • Canva (pour faire les diaporamas)
  • Jira (pour gerer l'avancement)

Les étapes du projet

Le développement s'est déroulé en 4 versions successives. Cliquez sur chaque version pour voir les détails.

Le serpent de taille fixe se déplace automatiquement vers la droite. L'utilisateur définit sa position initiale via les coordonnées (x, y). Le jeu commence ensuite avec un affichage dynamique dans le terminal.

Le serpent peut se déplacer dans toutes les directions avec les touches ZQSD. Le demi-tour est interdit. Le serpent peut se croiser et sortir de l’écran.
