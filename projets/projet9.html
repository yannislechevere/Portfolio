Serveur web et développement Apache

Ce projet a pour objectif d'installer et configurer des services réseaux permettant de développer et de déployer des applications informatiques communicantes. L’objectif étant de répondre à un besoin exprimé par un client.

Compétence n°9 : Installation de services réseau

Ce projet est en lien avec la compétence 9 du programme du BUT Informatique : Installer, configurer, mettre à disposition, maintenir en conditions opérationnelles des infrastructures, des services et des réseaux et optimiser le système informatique d’une organisation.

Les outils utilisés :

  • Langage PHP
  • Langage HTML
  • Langage CSS
  • Google Docs (pour rédiger les comptes rendus)
  • Visual Studio Code (pour éditer le code)

Les étapes du projet

Ce projet avait pour but de comprendre le fonctionnement d'un serveur web et de faire un petit site web avec un système de connexion très simple pour utiliser un serveur web.

Dans cette partie, il fallait rédiger un rapport avec des captures d'écran de notre travail. Ce rapport devait répondre à une série de travaux pratiques donnés.

C'est la mise en place d'un site web (sur le thème de Naruto). Ce site comporte une partie connexion/inscription et s'appuie sur le serveur Apache.
