Le jeu du Snake

Ce projet a été réalisé individuellement en C, sous forme d'une version console. L'objectif est d'implémenter plusieurs versions du jeu Snake, chacune ajoutant des fonctionnalités progressives.

Compétence n°1 : Implémentation d'un besoin client

Ce projet s'inscrit dans la compétence 1 du programme du BUT Informatique : développer une solution informatique adaptée à un besoin client.

Les outils utilisés :

  • Langage C (pour programmer les versions)
  • Visual Studio Code (pour éditer le code)
  • Doxygen (pour la documentation du code)
  • Google Sheets (pour les cahiers de tests)

Les étapes du projet

Le développement s'est déroulé en 4 versions successives et une documentation de la derniere version. Cliquez sur chaque version pour voir les détails.

Le serpent de taille fixe se déplace automatiquement vers la droite. L'utilisateur définit sa position initiale via les coordonnées (x, y). Le jeu commence ensuite avec un affichage dynamique dans le terminal.

Le serpent peut se déplacer dans toutes les directions avec les touches ZQSD. Le demi-tour est interdit. Le serpent peut se croiser et sortir de l’écran.

Le plateau de 80x40 est délimité par des bordures. Le serpent meurt s’il touche un mur. Quatre pavés d’obstacles sont générés aléatoirement dans le cadre sans toucher les bords.

Le serpent mange des pommes représentées par ‘6’. À chaque pomme mangée : le score augmente, la taille du serpent s’allonge et la vitesse s’accélère. À 10 pommes mangées, la partie est gagnée. Le serpent peut traverser le plateau via des passages sur chaque bord.
