Application de dance

Ce projet consistait à réaliser une application Java de gestion (implémentation d'une partie de l'application de danse concernant la gestion du plan des tables pour un gala).

Compétence n°7 : Développement d’une application

Ce projet est en lien avec la compétence 7 du programme du BUT Informatique : Développer, c'est-à-dire concevoir, coder, tester et intégrer une solution informatique pour un client.

Les outils utilisés :

  • Langage Java
  • JavaFX (pour l'interface graphique)
  • Eclipse (IDE pour développer et lancer le code)
  • Scene Builder (pour créer les fichiers FXML)

Le projet

Le projet consiste à développer la fonctionnalité de gestion des tables pour un gala de danse : création des tables, attribution des places, et export du plan.
